Image Processing Assignment (Q1 – Q12)

Overview
This project implements a series of fundamental image processing techniques using Python, NumPy, OpenCV, and Matplotlib.
Each question (Q1–Q12) is implemented in a separate Python file, covering both spatial and frequency domain operations.

Topics Covered
Intensity Transformations (Gamma Correction, Contrast Stretching)
Histogram Analysis & Equalization
Image Enhancement (Sharpening, Filtering)
Noise Reduction Techniques
Edge Detection (Sobel, Canny)
Image Resizing (Nearest, Bilinear Interpolation)
Spatial vs Frequency Domain Filtering
Homomorphic Filtering (Illumination Correction)
